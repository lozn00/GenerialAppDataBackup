# -
无需安装任何框架，无需root,目前只支持安卓机，后续可以考虑 适配ios.

开机usb调试 无线连接调试，然后图片识别，通用方案。图形化配置一下选框 区域识别的内容就行。

# 用途

主要是方便备份聊天记录。
备份关注人信息。
针对作废的账号方便快速迁移联系人。

但是传统的聊天记录 备份是无法查看的，非法手段本身不合法，这个只需要adb开启无线调试 然后电脑连接上手机的ip就行。

# 实用性

完全通用的，适用于任何软件上提取采集信息存档。

# 可配置

配置之后才能正确的识别是列表页还是详情页。



# 适用页面结构
- 最近聊天列表->聊天详情-（点头像）个人主页

可收集到 用户人昵称   聊天内容   用户名个人主页信息(id提取）

- 关注列表-（点头像）个人主页


可收集到 用户人昵称      用户名个人主页信息(id提取）
